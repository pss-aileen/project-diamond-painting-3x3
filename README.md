# DIAMOND PAINTING MINI
ダイヤモンドアートというものが気になっていた時期がありました。少し高いものだったため、「買うかどうか決めるためにおためし体験ができたらいいな」という理由で、Webブラウザで擬似ダイヤモンドアートを作ってみました。

## 概要
9つランダムで色のついたダイヤが生成されるのでそれを上の枠に綺麗に並べるゲーム（？）です

URL: https://pss-aileen.github.io/project-diamond-painting-3x3/

## 使い方、楽しみ方（？）
生成されたダイヤを上の枠になるべく綺麗に配置します。
